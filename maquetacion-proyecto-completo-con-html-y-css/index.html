Noticias de front-end

a

Titulo de la entrada Una página web está compuesta por dos partes: el front end y el back end. El front end es la parte visible para los usuarios de una página, de ahí su importancia. A causa de esto también es conocido como “el lado del cliente”.

[20-12-2020. Cristina Pérez Huerta]
a

Titulo de la entrada Se trata de una parte que conecta todo lo que tiene el back end, sus bases de datos, con el usuario para que pueda navegar por la web de manera fácil y sin que haya errores de servidor. Aunque a priori parece que solo es una parte visual carente de importancia, en realidad tiene un papel muy importante en el desarrollo de una web.

[20-12-2020. Cristina Pérez Huerta]
a

Titulo de la entrada Piensa que en el mundo actual existen muchos e-commerce que viven de sus negocios online. En consecuencia, la interacción entre el back end y front end debe ser fluida y óptima para que la experiencia de usuario sea la mejor.

[20-12-2020. Cristina Pérez Huerta]
a

Titulo de la entrada Si un posible cliente llega a tu web y la considera poco intuitiva o tal vez produce errores, es posible que decida abandonarla. Esto es lo que genera que el trabajo de un desarrollador web sea tan importante.

[20-12-2020. Cristina Pérez Huerta]
